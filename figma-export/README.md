# CIBA Portfolio Figma Export

This folder is a Figma handoff package for the current portfolio website.

## Best Editable Import

1. Open Figma.
2. Go to `Plugins > Development > Import plugin from manifest...`.
3. Select:
   `C:\Users\15340\Documents\33333\figma-export\figma-plugin\manifest.json`
4. Run `CIBA Portfolio Import`.
5. The plugin will create editable Figma frames for:
   - Home / Desktop
   - Home / Mobile
   - Works / Scroll System
   - Project Detail / X.WHEEL
   - Wake Up / Replica Map
   - About / Statement CV Contact

The generated frames use editable Figma text, rectangles, lines, and vector-like placeholders. Use them for layout editing.
